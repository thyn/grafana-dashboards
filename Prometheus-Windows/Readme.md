[Windows-Base](window-base.json)

Contains base metrics for Windows Grafana base on Prometheus and wmi_exporter:

- Cpu: base metrics
- Network under windows: 
  - By interfaces
  - By TCP Socket states
  - By TCP Segments=
- Disks:
  - By Drive
  - IO and Latency
- By Process:
  - Threads
  - CPU Time
  - IO
